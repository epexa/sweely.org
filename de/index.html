Häufig gestellte Fragen

Wer ist Freiwilliger?
Ein Benutzer, der eine Einzelperson ist und seine Unterstützung (Arbeit) in einem sozialen Projekt für eine Belohnung leistet.
Wer ist der Sponsor?
Eine juristische oder natürliche Person, die unwiderruflich Waren oder Dienstleistungen erbringt oder zur vorübergehenden Verwendung bei einer Prämie.
Was ist eine gemeinnützige Organisation?
Eine registrierte Organisation, deren Haupttätigkeit die Schaffung sozialer Projekte und Initiativen ist.
Was sind Ressourcen?
Waren und Dienstleistungen, die von Sponsoren für Prämien bereitgestellt werden. Sie befinden sich im "Store".
Was ist eine Belohnung?
Dies sind Ergebnisse von der Webplattform von Sweely.org. Jede Ressource, die sich im Abschnitt „Store“ befindet, hat einen eigenen Wert. Sie können Belohnungen nur auf unserer Plattform Sweely.org sammeln und ausgeben.
Wie verdient man Belohnungen?

Wenn Sie ein Freiwilliger sind

Erhalten Sie Belohnungen, die Sie bei NGOs in sozialen Projekten mitmachen können.

Wenn du NPO bist

Sie können Belohnungen verdienen, indem Sie Ihr soziales Projekt oder Ihre Initiative umsetzen. Nach Abschluss des Projekts erhalten Sie eine durch die Formel berechnete Belohnung.

Die Höhe der Belohnung hängt von Ihrer Aktivität auf der Sweely-Plattform und der Erstellung von Projekten ab, von der Anzahl der beteiligten Freiwilligen (Skala), dem Erfolg des Projektabschlusses sowie von den Rückmeldungen der Freiwilligen.

Wenn Sie ein Sponsor sind

Sie können Belohnungen verdienen, indem Sie Ihre Ressourcen im Abschnitt „Store“ anbieten und veröffentlichen. Nach dem Kauf durch einen anderen Benutzer oder andere Benutzer der Plattform erhalten Sie die von Ihnen auf Ihrem Konto angegebene Prämie.

Wie gebe ich Belohnungen aus?
Sie können Ihre verdienten Belohnungen ausgeben:

Kaufen Sie eine Ressource im Abschnitt "Store". Gehen Sie zur Seite "Store" und wählen Sie die gewünschte Ressource innerhalb des Betrags der Belohnungen aus, die Sie auf Ihrem Konto haben.

Übersetzen Sie Belohnungen in ein soziales Projekt oder eine Organisation, die Ihnen gefällt (um Wohltätigkeitsorganisationen zu bieten).

Was kostet die Nutzung der Plattform?
Es ist absolut kostenlos für alle Benutzer der Plattform. Es gibt keine versteckten Gebühren und Provisionen.
